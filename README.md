#### 終端：母星觀測站 (Terminal: Homeworld Observatory)

## 燈塔計畫 (Project Lighthouse)

本專案為「網頁設計課程」之期末專題作品。這是一個基於原創世界觀「燈塔計畫」所建構的沉浸式科幻體驗網頁。透過純前端技術（HTML/CSS/JS），模擬了一個末日後的母星觀測終端系統，使用者可透過此介面監測環境數據、檢視異質礦物資料庫以及進行深層地質生態探勘。

## 專案概覽

本網頁旨在展示不依賴大型框架下的原生網頁開發能力，重點在於視覺動效的呈現與世界觀的氛圍營造。系統包含三大核心模組：

* **主控台 (Dashboard)**：模擬觀測站窗口，具備視差滾動與桌面交互功能。
* **地脈資料庫 (Veins)**：展示虛構礦物與高維元素的詳細設定，包含複雜的行星軌道動畫。
* **深淵探勘 (Ecological)**：模擬垂直下潛的作業系統，展示從地殼至內核的地層資訊。

## 技術亮點與實作細節

本專案完全使用原生語法開發，強調 CSS 動畫與 JavaScript DOM 操作的結合。

### 視覺渲染引擎

* **CRT 顯像管風格**：利用 CSS 濾鏡與漸層（`linear-gradient`）繪製掃描線（Scanlines）與暗角效果，配合微幅的 `opacity` 動畫模擬老式螢幕的訊號抖動。
* **3D 視差位移 (Parallax Effect)**：在 `index.html` 與 `veins.html` 中，透過 JavaScript 監聽滑鼠座標（`mousemove`），動態計算不同圖層（如窗框、背景星空、前景儀表）的 `translate3d` 與 `rotate` 數值，營造強烈的空間縱深感。
* **高科技介面動效**：大量使用 `transform: skew` 與 `backdrop-filter: blur` 製作非對稱且具備毛玻璃效果的 UI 組件。

### CSS 動畫系統

* **天體軌道模擬**：在 `veins.html` 中，不使用 3D 引擎，而是透過純 CSS 的 `@keyframes` 與巢狀 `div` 結構（`orbit-container`），透過精確計算的 `rotate` 與 `translateX` 數值，實現多層次、不同速率的行星公轉與自轉效果。
* **粒子與環境特效**：手刻 CSS 動畫來模擬咖啡杯的蒸汽上升、菸灰缸的煙霧飄散以及背景星空的呼吸燈效果。
* **動態文字亂碼**：導航選單實作了滑鼠懸停（Hover）時的解碼特效，透過 JavaScript 隨機替換字元，模擬資料傳輸不穩定的科幻感。

### 交互邏輯

* **深度捲動監測**：在 `ecological.html` 中，利用監聽捲動事件來驅動左側的深度計數器與右側的地層資訊卡片切換。
* **純 CSS 狀態管理**：大量運用 `<input type="checkbox">` 與 `<input type="radio">` 配合 CSS 選擇器（`input:checked ~ .content`）來實作彈窗開關、分頁切換與開發者模式（Developer Mode），減少對 JavaScript 的依賴。

## 頁面功能介紹

### index.html (觀測站首頁)

* **環境監控**：背景包含動態視訊疊加層，展示觀測窗外的荒原景色。
* **桌面交互**：桌面上放置的文件、平板與咖啡杯皆為可互動元件，點擊可連結至外部 Google Docs 企劃書或觸發特效。
* **操作指引**：內建「OPERATIONS MANUAL」彈窗，以輪播圖形式引導使用者操作。

### veins.html (地脈資料庫)

* **礦物光譜儀**：中央呈現複雜的動態魔法陣與軌道模型。
* **詳細日誌 (Logs)**：收錄 10 種一般礦物（如：燼紅石、天青浮岩）與 5 種高維元素（如：艾特諾、費濟姆）的詳細學術設定。
* **視覺切換**：點擊列表可觸發右側詳細資料卡的滑入動畫與粒子特效。

### ecological.html (深淵探勘)

* **NetherDiver OS**：模擬鑽探機具的作業系統介面，包含開機自檢代碼流（Code Log）。
* **地層剖面**：隨捲動展示地殼、上地函、下地函、外核至內核的環境數據（溫度、壓力、威脅等級）。
* **警告系統**：到達底部時觸發「負荷過載」的視覺警告效果。

## 檔案結構說明

* `index.html` - 網站入口與主控台
* `veins.html` - 礦物與元素資料庫
* `ecological.html` - 地質生態垂直捲動頁面
* `index/` - 存放首頁相關的 CSS (`decoration.css`, `index-styles.css` 等)、JS 與圖片資源
* `Veins/` - 存放地脈頁面的專用樣式 (`log.css`, `celestial motion.css`) 與資源
* `abyss/` - 存放生態頁面的專用樣式與資源

## 安裝與執行

1. 複製此儲存庫至本地端：
`git clone https://github.com/YourUsername/Project-Lighthouse.git`
2. 直接使用現代瀏覽器（建議使用 Chrome 或 Edge 以獲得最佳 CSS 渲染效能）開啟 `index.html` 檔案。
3. **注意**：為了體驗完整的沉浸式音效，請與頁面互動以允許瀏覽器自動播放背景音樂。

## 世界觀設定 (Lore)

本專案基於「燈塔計畫」世界觀。在母星環境崩潰後，觀測站成為了紀錄星球最後生機的終端。

* **異質礦物**：紀錄了因維度崩塌或地質變異產生的特殊礦石，例如能鎖定熱能的「燼紅石」或具備相位不穩定性的「費濟姆」。
* **地層生態**：從地表的荒原向下延伸，紀錄了在地底深處依靠地脈能量與化學合成作用生存的獨特生態系。
